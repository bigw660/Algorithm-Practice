Straight forward approach:
```python
def sortArrayByParity(self, A):
        even, odd = [], []
        
        for x in A:
            if x%2 == 0:
                even.append(x)
            else:
                odd.append(x)
        
        return even + odd
```
Time: `O(n)`
Space: `O(n)`

One-liner:
```python
def sortArrayByParity(self, A):
        return sorted(A, key=lambda x:x%2)
```
Time: `O(n log n)`
Space: `O(1)`

**Remark:** there is a better approach by 2-pointer, which has time `O(n)` and space `O(1)`.
