This is similar to *#94. Binary Tree Inorder Traversal*. The only difference is to reverse the ooutput order.
**Post-order:**
```python
def postorderTraversal(self, root):
        stack, res = [], []
        cur = root

        while stack or cur:
            if cur:
                stack.append(cur)
                res.append(cur.val)
                cur = cur.right
            else:
                cur = stack.pop()
                cur = cur.left

        return res[::-1] # need to reverse the order
```

**In-order:**
```python
def inorderTraversal(self, root):
        stack, res = [], []
        cur = root
        
        while stack or cur:
            if cur:
                stack.append(cur)
                cur = cur.left
            else:
                cur = stack.pop()
                res.append(cur.val)
                cur = cur.right
                
        return res
```
