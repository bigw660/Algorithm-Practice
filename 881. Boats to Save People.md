The basic idea is greedy.
```python
def numRescueBoats(self, people, limit):  
        # stack + 2-sum       
        stack = []
        for x in people:
            if x < limit:
                stack.append(x)
        
        stack.sort()
        p, q = 0, len(stack)-1
        pair = 0
        while p < q:
            if stack[p]+stack[q] <= limit:
                pair += 1
                p += 1
                q -= 1
            else:
                q -= 1
    
        return len(people) - pair
```
